# Aurora Career Copilot

AI-powered career coaching and learning assistant for developers.

## Features

- DSA Learning Roadmap
- Internship & Job Preparation
- Project Ideas & Portfolio Building
- Hackathon & Contest Discovery
- Personal Branding (GitHub & LinkedIn)

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- PWA (Progressive Web App)
- Ollama (Local AI)

## Getting Started

### Prerequisites

- Node.js 18+
- Ollama (optional, for local AI)

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

### Using with Ollama (Optional)

1. Install Ollama: https://ollama.ai
2. Pull the model: `ollama pull llama3.2`
3. Start Ollama: `ollama serve`
4. The app will automatically connect to local Ollama

### Deployment

Deploy to Vercel:

```bash
npm run build
```

Then connect your GitHub repository to Vercel for automatic deployments.

## PWA Installation

- **Android**: Chrome → Menu → Add to Home Screen
- **iOS**: Safari → Share → Add to Home Screen
